================================================================================
USAGE INSTRUCTIONS
================================================================================

Quick Start:
1. cp .env.example .env
2. Edit .env and add TELEGRAM_BOT_TOKEN
3. ./quickstart.sh

Or with Docker:
1. cp .env.example .env
2. Edit .env and add TELEGRAM_BOT_TOKEN
3. docker-compose up -d

Commands in Telegram:
- /start - Initialize bot
- /help - Get help
- Ofertas - Search jobs
